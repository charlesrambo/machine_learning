Objective¶

Our objective for this exercise is quite limited. We will use Bayesian optimization to estimate the optimal hyperparamters for XGBoost and Neural Network models. We will add noise to the graph of $y = \sin(2\pi x)$. Then we write the average sum of square error from cross validation as a function of our hyperparameters $\theta$. That is, we will consider $$ f: \theta \mapsto \text{average sume of square error given }\theta. $$ We will estimate $f$ using a Gaussian process $\hat{f}$. Minimize this estimate $\hat{f}$. Resample from $f$ at $$ \theta^* = \text{arg}\displaystyle\min_\theta \hat{f}(\theta). $$ Then repeate until we reach the maximum iterations or the process converges. This process is less accurate but less computationally intensive compared to something like grid search.

Generating the data¶

As mentioned previously, we will take the graph of $y = \sin(2\pi x)$ and then add noise. In particular, we will map $$ y_i \mapsto y_i + \epsilon_i $$ where $\epsilon_i\sim{\mathcal{N}(0, 0.1^2)}$ and $\epsilon_i$ is independent from $\epsilon_j$ for $i\neq j$. We will generate $N = 1 \times 10^5$ values because neural networks require a large amound of data to be effictive predicting models.

# Define objective function
def obj_xgb(parameters):
    parameters = parameters[0]
    score = -cross_val_score(
        xgb.XGBRegressor(
            learning_rate=parameters[0],
            gamma = parameters[1],
            max_depth = int(parameters[2]),
             n_estimators =int(parameters[3])),
        x_train, y_train, scoring = 'neg_mean_squared_error').mean()
    score = np.array(score)
    return score

def obj_nn(parameters):
    parameters = parameters[0]
    score = -cross_val_score(
        nn.MLPRegressor(alpha = parameters[0], hidden_layer_sizes = int(parameters[1])),
        x_train, y_train, 
        scoring = 'neg_mean_squared_error').mean()
    score = np.array(score)
    return score
